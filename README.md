# 📰 AI Fake News Detection System

An AI-powered web application that detects whether a news article or headline is **Real or Fake** using **Machine Learning and Natural Language Processing (NLP)**.

This system analyzes the writing patterns of news text and predicts its authenticity with a confidence score.

---

## 🚀 Features

* Detects **Fake vs Real News**
* Analyze **news headlines or full article text**
* Analyze **news from article URLs**
* Shows **confidence score**
* **Prediction analytics chart**
* **History of predictions**
* Clean **React UI dashboard**

## 🖥️ Installation

Clone the repository:

```bash
git clone https://github.com/VishalSudhaArul/Fake-News-Detection.git
cd Fake-News-Detection
```

---

### Backend Setup

```bash
cd Backend
pip install -r requirements.txt
python app.py
```

Server runs on:

```
http://127.0.0.1:5000
```

---

### Frontend Setup

```bash
cd Frontend
npm install
npm run dev
```

Frontend runs on:

```
http://localhost:5173
```
